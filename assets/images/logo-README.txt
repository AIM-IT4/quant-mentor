# Logo Instructions

Please save your QuantMentor logo as: `assets/images/logo.png`

This logo will be used as:
- Website favicon (browser tab icon)
- Open Graph image (when sharing on social media)
- PWA icon (if you add app functionality)
- Navbar logo (optional)

Recommended specifications:
- Size: 512x512px or larger
- Format: PNG with transparency
- Background: Transparent or dark (matching your website theme)

Current placeholder: The logo is referenced in index.html and will display once you upload the file to assets/images/logo.png
